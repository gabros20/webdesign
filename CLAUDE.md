# Claude Code repository guide

Read [AGENTS.md](AGENTS.md) before changing runtime behavior and
[CONTRIBUTING.md](CONTRIBUTING.md) before preparing a release. Keep `webdesign` focused on website
direction, appearance-producing implementation, and rendered review; hand general engineering,
accessibility, SEO, growth, and lifecycle composition to their owning capabilities.

Run `scripts/check-sync` after every runtime, routing, metadata, documentation, or evaluation
change. Do not claim visual completion unless the relevant rendered evidence was inspected.
